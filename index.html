mysql索引

查询数据时查询索引列。虽然可以提高查询速度，但是会影响插入操作，也占用物理空间。
索引有两种存储类型，包括btree和hash，btree为默认索引类型

创建数据表时创建索引

普通索引

查看建表语句，如图 该表结构索引为id

唯一性索引

需要使用UNIQUE参数进行约束
（有时不能实现优化查询的目的）

全文索引

适用于char varchar text类型，使用fulltext参数约束
(fulltext key)

单列索引

只需指定字段名，可指定索引字段长度，提高查询速度

多列索引

查询条件满足第一个字段才会使用索引

空间索引

myisam类型表，设置spatial参数,类型应为geometry

在已存在的数据表中创建索引

create （参数）index index_name on table_name (字段名称)

alter table table_name add (参数) index index_name(字段名称)

删除索引

drop index index_name on table_name;

title: LAMP
date: 2018-05-24

====确定存在linux系统，新手第一次安装可参考以上网址 ====

可参考 virtual box安装centos6版本的Linux系统

www服务器

使用的语言多数为超文本标记语言（HTML）

支持的协议为 http（明码），https（加密）

浏览器应支持制定标准（W3C） –如firefox chrome
附：IE由于是内核编译，缺乏安全性

URL是浏览器与服务器获取与接收的定位网址（协议：//ip.目录资源)

LAMP是WWW服务器类型，包括系统（Linux），平台（Apache），数据库（mysql），程序（php，perl，asp，jsp，cgi网页程序语言）
附：php，perl，asp，jsp，cgi都是网页程序语言

（笔者使用centos自带的apache，php，mysql）
须知，php是apache执行的一个模块，另外php也需mysql支持

yum install httpd mysql mysql-server php php-mysql

  • Apache
    配置文件为 /etc/httpd/conf/httpd.conf
    可设置系统管理员邮箱 ServerAdmin选项
    对中文编码的设置 default charset

  • PHP
    配置文件为 /etc/php.ini
    修改ignore_repeated_errors = On; ignore_repeated_errors = On;
    以上，将存在的重复日志数据忽略

启动服务
/etc/init.d/httpd start
/etc/init.d/httpd configtest (测试配置文件的语法错误）
chkconfig httpd on

附：Apache携带的apachectl程序启动与关闭服务
/usr/sbin/apachectl start
/usr/sbin/apachectl stop

  • mysql
    启动：/etc/init.d/mysql start
    chkconfig mysqld on
    手动连接mysql数据库 ：mysql -u root -p
    （首次登陆不用密码，可用mysqladmin配置密码：mysqladmin -u root password‘xxx’）

防火墙与selinux设置
iptables -A INPUT -t TCP -i eth0 –dport 80 –sport 1024：65534 -j ACCEPT
附：请勿开放3306数据库端口

setsebool -P httpd_can_network_connect=1

有错误的地方请指导，邮箱985207502@qq.com

服务器搭建

ftp

ftp服务器搭建（Linux）

相关软件 vsftpd

如果是第一次使用，在命令行输入 yum install vsftpd

服务器设置

1.服务器环境设置

服务器启动模式:listen=YES(默认为stand daemon，如果改用super daemon应启动xined，listen=NO）
欢迎信息
配置banner_file=/etc/vsftpd/welcome.txt
建立欢迎界面
vim /etc/vsftpd/welcome.txt

2.与实体用户较相关设置

3.匿名用户登录设置

匿名用户只能在/var/ftp浏览（可在此目录放置文件）

4.关于系统安全方面设置

==防火墙加入所需要的FTP模块==
vim /etc/sysconfig/iptables-config
IPTABLES_MODULES=”ip_nat_ftp ip_conntrack_ftp”
开启iptables /etc/init.d/iptables restart

==兼顾主动式与被动式连接==
iptables -A INPUT -p TCP -i eth0 –dport 21 –sport 1024:65534 -j ACCEPT
iptables -A INPUT -p TCP -i eth0 –dport 65400:65410 –sport 1024:65534 -j ACCEPT

附：
ftp账号添加命令：useradd passwd
知道自己ftp服务器的ip地址： ifconfig

登录ftp

ftp –> open [服务器ip地址]

存在问题

默认一般用户登录无法取得用户主目录，是SElinux问题，可通过修改规则：
setsebool -P ftp_home_dir=1
