django项目的环境预先安装

发表于 2018-07-21 | 分类于 python

基于LINUX系统

Python3下载
笔者运行的环境（centos6.9），参考多个博客，打算记录一下

python3 

更新依赖包  
  `yum -y install zlib-devel bzip2-devel openssl-devel ncurses-devel sqlite-devel readline-devel tk-devel gdbm-devel db4-devel libpcap-devel xz-devel  gcc openssl openssl-devel libcurl-devel`  

下载Python程序包  
wget http://python.org/ftp/python/3.6.6/Python-3.6.6.tar.xz --no-check-certificate

编译安装  
tar xvf Python-3.6.6.tar.xz

cd Python-3.6.6  
./configure
make all   
make install   

建立软连接
rm -rf /usr/bin/python  
ln -s /usr/local/bin/python3.6 /usr/bin/python

pip下载（以后可以下载Python模块） 

1、安装pip前需要前置安装setuptools  
wget --no-check-certificate  https://pypi.python.org/packages/source/s/setuptools/setuptools-19.6.tar.gz#md5=c607dd118eae682c44ed146367a17e26

tar -zxvf setuptools-19.6.tar.gz

cd setuptools-19.6

python setup.py build

python setup.py install

2、安装pip  
wget --no-check-certificate  https://pypi.python.org/packages/source/p/pip/pip-8.0.2.tar.gz#md5=3a73c4188f8dbad6a1e6f6d44d117eeb

tar -zxvf pip-8.0.2.tar.gz

cd pip-8.0.2

python setup.py build

python setup.py install

ok，我们终于完成了整个默认环境为python3的安装。

接下来，安装虚拟环境

pip install virtualenv

然后可以使用virtualenv命令创建python虚拟环境：virtualenv [虚拟环境名称]。

进入虚拟环境：source venv/bin/activate

在虚拟环境中下载相应的pip需求模块，可以使用pip install -r filename

笔者使用的django基于python3的1.10.1版本

创建django项目 django-admin startproject blog

运行项目 python manage.py runserver xxxx:8000

创建应用 python manage.py startapp myblog
