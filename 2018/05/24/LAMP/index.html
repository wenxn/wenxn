LAMP

发表于 2018-05-24 | 分类于 搭建

====确定存在linux系统，新手第一次安装可参考以上网址 ====

可参考 virtual box安装centos6版本的Linux系统

www服务器

使用的语言多数为超文本标记语言（HTML）

支持的协议为 http（明码），https（加密）

浏览器应支持制定标准（W3C） –如firefox chrome
附：IE由于是内核编译，缺乏安全性

URL是浏览器与服务器获取与接收的定位网址（协议：//ip.目录资源)

LAMP是WWW服务器类型，包括系统（Linux），平台（Apache），数据库（mysql），程序（php，perl，asp，jsp，cgi网页程序语言）
附：php，perl，asp，jsp，cgi都是网页程序语言

（笔者使用centos自带的apache，php，mysql）
须知，php是apache执行的一个模块，另外php也需mysql支持

yum install httpd mysql mysql-server php php-mysql

  • Apache
    配置文件为 /etc/httpd/conf/httpd.conf
    可设置系统管理员邮箱 ServerAdmin选项
    对中文编码的设置 default charset

  • PHP
    配置文件为 /etc/php.ini
    修改ignore_repeated_errors = On; ignore_repeated_errors = On;
    以上，将存在的重复日志数据忽略

启动服务
/etc/init.d/httpd start
/etc/init.d/httpd configtest (测试配置文件的语法错误）
chkconfig httpd on

附：Apache携带的apachectl程序启动与关闭服务
/usr/sbin/apachectl start
/usr/sbin/apachectl stop

  • mysql
    启动：/etc/init.d/mysql start
    chkconfig mysqld on
    手动连接mysql数据库 ：mysql -u root -p
    （首次登陆不用密码，可用mysqladmin配置密码：mysqladmin -u root password‘xxx’）

防火墙与selinux设置
iptables -A INPUT -t TCP -i eth0 –dport 80 –sport 1024：65534 -j ACCEPT
附：请勿开放3306数据库端口

setsebool -P httpd_can_network_connect=1
