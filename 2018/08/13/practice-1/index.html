1.二维数组中的查找

题目：二维数组中，每行从左到右递增，每列从上到下递增，给出一个数，判断它是否在数组中

思路：通过一个一个比较，我的方法是由最后一行的第一个开始比较，因为是有序序列，可以简单做判断

假设rows, cols = len(array), len(array[0])，表示array的长度为数组的行数，array[0]的长度为数组列数

查看代码：

# -*- coding:utf-8 -*-
class Solution:
# array 二维列表
def Find(self, target, array):
    rows, cols = len(array), len(array[0])
    row, col = rows - 1, 0
    while row >= 0 and col <= cols - 1:
        if array[row][col] == target:
               return True
        elif array[row][col] > target:
            row -= 1
        else:
            col += 1
    return False

ps：查看一个数是否在列表上可以试试看 in 标识符

3.从尾到头打印单链表

思路：新建一个空列表，加入元素，再反向排序，返回

class Solution:

def printListFromTailToHead(self, listNode):
# write code here

    listNode = listNode
    while listNode:
        d.append(listNode.val)

4.重建二叉树

要求：用前序和中序遍历结果构建二叉树，遍历结果中不包含重复值
思路：前序的第一个元素是根结点的值，在中序中找到该值，（使用一下内置方法index找出对应的字符串，返回索引），中序中该值的左边的元素是根结点的左子树left，右边是右子树right，然后递归的处理左边和右边

class Solution:
# 返回构造的TreeNode根节点
def reConstructBinaryTree(self, pre, tin):
    if not pre or not tin:
        return None
    index = tin.index(pre[0])
    left = tin[0:index]
    right = tin[index+1:]
    root = TreeNode(pre[0])
    root.left = self.reConstructBinaryTree(pre[1:1+len(left)], left)
    root.right = self.reConstructBinaryTree(pre[-len(right):], right)
    return root

ps：在递归处理时，应该是调用自身的方法，不是用self做参数。里边的参数分别是前序遍历和中序遍历 

5.用两个栈实现队列

要求：用两个栈来实现一个队列，完成队列的Push和Pop操作。 队列中的元素为int类型。
思路：栈A用来作入队列;栈B用来出队列，当栈B为空时，栈A全部出栈到栈B,栈B再出栈（即出队列）

# -*- coding:utf-8 -*-
class Solution:
    #初始化两个栈
    def __init__(self):
        self.stackA = []
        self.stackB = []
    #A入队列     
    def push(self, node):
        self.stackA.append(node)
    #B出队列，A如果还有元素，加入B，再出     
    def pop(self):
        if self.stackB:
            return self.stackB.pop()
        elif not self.stackA:
            return None
        else:
            while self.stackA:
                self.stackB.append(self.stackA.pop())
            return self.stackB.pop()

ps：在做题之前没想过初始化栈，对队列不太理解。主要算法难在b队列先出，a队列后出，表明队列有先进先出，与栈不同。
