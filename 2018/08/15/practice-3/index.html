11.二进制1的个数

题目描述：
输入一个整数，输出该数二进制表示中1的个数。其中负数用补码表示。
思路：
（来自网友的看法，觉得很巧妙，可以学一学）
如果一个整数不为0，那么这个整数至少有一位是1。如果我们把这个整数减1，那么原来处在整数最右边的1就会变为0，原来在1后面的所有的0都会变成1(如果最右边的1后面还有0的话)。其余所有位将不会受到影响。

举个例子：一个二进制数1100，从右边数起第三位是处于最右边的一个1。减去1后，第三位变成0，它后面的两位0变成了1，而前面的1保持不变，因此得到的结果是1011.我们发现减1的结果是把最右边的一个1开始的所有位都取反了。这个时候如果我们再把原来的整数和减去1之后的结果做与运算，从原来整数最右边一个1那一位开始所有位都会变成0。如1100&1011=1000.也就是说，把一个整数减去1，再和原整数做与运算，会把该整数最右边一个1变成0.那么一个整数的二进制有多少个1，就可以进行多少次这样的操作。

# -*- coding:utf-8 -*-
class Solution:
def NumberOf1(self, n):
# write code here
        c = 0
        if n<0:
            #对负数做处理，为32位
            n = n&0xffffffff
        while n:
            c += 1
            n &= n-1
        return c

因为python的int是无线精度的，c++的int是32位的，所以python的负数相当于前面有无限个1，要对python的负数做处理

12.数值的整数次方

题目描述：
给定一个double类型的浮点数base和int类型的整数exponent。求base的exponent次方。

思路：考虑整数为正为负为零情况，结果可以用循环*=base返回

# -*- coding:utf-8 -*-
class Solution:
    def Power(self, base, exponent):
        flag = 0
        if base == 0:
            return False
        if exponent == 0:
            return 1
        if exponent < 0:
            flag = 1
        result = 1
        for i in range(abs(exponent)):
            result *= base
        if flag == 1:
            result = 1 / result
        return result

ps：此处用flag标识是否倒装，abs（）函数是取绝对值，for循环是为了得到result结果，总函数的return值是与def对应

13.调整数组顺序使奇数位于偶数前面

题目描述：
输入一个整数数组，实现一个函数来调整该数组中数字的顺序，使得所有的奇数位于数组的前半部分，所有的偶数位于数组的后半部分，并保证奇数和奇数，偶数和偶数之间的相对位置不变。

思路：制作两个空列表，依次遍历写进去，不影响相对位置，也方便奇偶分组。

# -*- coding:utf-8 -*-
class Solution:
def reOrderArray(self, array):
        ji,ou=[],[]
        for i in array:
            ji.append(i) if i%2==1 else ou.append(i)
        return ji+ou

ps：这里用到if三分选择，当然也可以使用简单的if语句

14.链表中倒数第k个结点

题目描述：输入一个链表，输出该链表中倒数第k个结点。

思路：建立空列表，写入结点（head），并非是结点值（head.val）。这一点注意一下,写入结点列表和从头打印的逻辑一样，接着函数值return[-k]表示倒数第k个值

class Solution:
    def FindKthToTail(self, head, k):
        # write code here
        res=[]
        while head:
            res.append(head)
            head=head.next
        if k>len(res) or k<1:
            return
        return res[-k]

ps：在调试过程中出现，“’NoneType’ object has no attribute ‘next’”，因为while判断条件有问题，要改成while head，假设空链表。
